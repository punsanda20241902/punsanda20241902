## Featured Projects
### 1) IPL Cricket Performance Analytics Dashboard
- Analysed **360 IPL matches (2018–2023)** to evaluate team performance, batting consistency, powerplay trends, and home vs away advantage.
- Built visualisations such as win heatmaps, strike rate vs average plots, and seasonal scoring trends.
- Key insight: **+5.6% rise** in average team scores over six seasons.

### 2) Retail Sales Analytics System
- Built an end-to-end multi-branch retail intelligence solution across **five cities**.
- Created a fully normalised relational database (**11 entities**), wrote advanced SQL analytics, and built a Python ETL pipeline.
- Delivered insights on branch revenue, CLV, product profitability, and inventory alerts.

### 3) Hospital Management Database System
- Designed a comprehensive healthcare database covering **13 interrelated entities** across the patient journey.
- Wrote SQL queries for operational KPIs (bed occupancy, doctor utilisation, revenue by department, length of stay, readmissions).
- Produced ERDs and Python reporting scripts to support data-driven decisions.

### 4) Hospital Patient Analytics Dashboard
- Analysed **1,200 patient records** across **8 departments (2023)**.
- Explored admission trends, billing patterns, patient outcomes, satisfaction, and readmissions.
- Identified elevated **60+ readmission risk (14–16%)** and suggested targeted interventions.

### 5) Household Finance & Budget Analytics
- Analysed **12 months** of household income/expenses across **10 categories**.
- Built cashflow trends, savings analysis (**20.1% average**), spending heatmaps, and budget vs actual comparisons.
- Key insight: housing accounted for **55%+** of total spend; provided budgeting recommendations.
